THE GAME
--------

It seems you have found me, welcome dear friend,
your productivity hath come to an end.

This game you have found is simple yet fun,
Add up all the numbers (yes every one).

To start enter numbers both upper and lower,
and choose the gap time whether faster or slower.

How many shall come? choose n with care.
as it sets the number of numbers found there

Press Start and behold as the screen comes alive,
A number appears - then another arrives.

No twin follows close, no copy allowed,
Each number shall differ, distinct and proud.

You’re given no sum, as they flash one by one,
so keep track yourself - let the mental math run.

When the final number has had its brief stay,
The game will grow silent and ask you to play.

“Input a guess,” the prompt appears,
then enter the total you've held for years.

If right, you are praised - correctness revealed,
If wrong, the true sum is gently unsealed.

Then back to the start, the cycle renews,
New bounds, new delays, new numbers to choose.

So focus, dear friend, let distraction be none:
This game is not speed — it’s remembering sum.
